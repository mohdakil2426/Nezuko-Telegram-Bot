## 1. Design System

### 1.1 Color Palette

| Token | Value | Usage |
|-------|-------|-------|
| **Primary-500** | `#6366F1` | Buttons, links, focus states |
| **Primary-700** | `#4338CA` | Hover states |
| **Success** | `#10B981` | Success messages, online status |
| **Warning** | `#F59E0B` | Warnings, pending states |
| **Error** | `#EF4444` | Errors, offline status |
| **Slate-950** | `#020617` | Page background (dark) |
| **Slate-900** | `#0F172A` | Card background |
| **Slate-400** | `#94A3B8` | Muted text |
| **Slate-50** | `#F8FAFC` | Primary text |

### 1.2 Typography

| Style | Size | Weight | Usage |
|-------|------|--------|-------|
| **Display** | 36px | Bold | Hero titles |
| **H1** | 28px | Semibold | Page titles |
| **H2** | 22px | Semibold | Card titles |
| **H3** | 18px | Medium | Section headers |
| **Body** | 14px | Regular | Default text |
| **Caption** | 12px | Regular | Labels, hints |

**Font Family**: Inter (Google Fonts)

### 1.3 Spacing & Borders

- **Spacing Base**: 4px (space-1 to space-32)
- **Border Radius**: 2px (sm), 6px (md), 8px (lg), 12px (xl)
- **Shadows**: Subtle elevation with slate-900
